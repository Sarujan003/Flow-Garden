# Flows Garden

*Cultivating the Wild — Established 2024*

Flows Garden is a premium, modern landing page designed to bridge the gap between concrete landscapes and thriving ecosystems. It serves as a platform for urban permaculture, community education, and garden design, fostering a community of wild-cultivators.

## ✨ Features

- **Dynamic Gallery**: Showcasing resilient plant species with an organic, modern UI.
- **Urban Permaculture**: Interactive sections detailing the philosophy of bridging urban spaces with nature.
- **Workshops & Events**: Information on community education and hands-on masterclasses.
- **Modern Design**: High-end aesthetics featuring glassmorphism, organic shapes, and smooth micro-animations.
- **Responsive Layout**: Seamless experience across all devices.

## 🛠️ Tech Stack

- **Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite 8](https://vitejs.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations**: [Motion](https://motion.dev/) (Framer Motion)
- **Icons**: [Lucide React](https://lucide.dev/)

## 🚀 Getting Started

### Prerequisites

- Node.js (Latest LTS recommended)
- npm

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd frontend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```

## 📂 Project Structure

- `src/components`: UI components (Cultivating, FeaturedFlora, Urban, HandsOn, FooterGarden).
- `src/assets`: Static assets and images.
- `src/App.tsx`: Main application assembly.
- `src/index.css`: Tailwind CSS entry point and custom design tokens.
